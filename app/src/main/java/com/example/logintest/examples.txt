 since you can tell the function what you want the response to be in terms of type(string/
 jsonArray/jsonObject)

 in my opinion, you should go in this order:

 if the server outputs JSON, then store as JSONObject/JSONArray, object is nice if you want to
 search for a key.

 otherwise just go for string



 public void postRequest() {
        HTTPHandler httpHandler = new HTTPHandler();
        JSONObject array = new JSONObject();
        try {
            array.put("username", "test");
        }catch (JSONException e){
            System.out.println("error");
        }

        String url = "https://lamp.ms.wits.ac.za/home/s2621933/test/post.php";
        JSONObject response = httpHandler.postRequest(url, array,JSONObject.class);
        TextView v = findViewById(R.id.textView);

        try {
            v.setText(response.getString("success"));
        }catch (JSONException e){
            System.out.println("screw you");
        }
    }
